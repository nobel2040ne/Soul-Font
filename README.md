# 🎨 Soul Font  
> ✨ *AI가 만들어낸 당신만의 손글씨 폰트*  
by **nobel2040ne** & **lukple1**  
A project from **Sejong Academy of Science and Arts**  

---

## 📌 개요  
**Soul Font**는 사용자의 손글씨를 기반으로 인공지능이 개인화된 폰트를 생성하는 프로젝트입니다.  

---

## 🧠 사용된 기술  
| 기술 | 설명 |
|------|------|
| 🐍 Python | 전체 AI 파이프라인 구현 |
| 🤖 PyTorch | 손글씨 학습 및 모델 훈련 |
| 🖼️ PIL / OpenCV | 이미지 전처리 및 처리 |
| 📝 FontForge + Node.js | TTF 폰트 자동 생성 |
| 🌐 Django | 웹 인터페이스 제공 |
| 🖥️ HTML/CSS/JS | 프론트엔드 구성 |

---

## 🛠️ 기능  
- ✏️ 손글씨 이미지 업로드  
- 🧽 손글씨 전처리 (crop, 보정 등)  
- 🎓 딥러닝 기반 학습 및 추론  
- 🧵 결과 이미지 패키징  
- 🧾 나만의 `.ttf` 폰트 파일 생성  
- 🌐 웹 기반 사용자 인터페이스 제공
